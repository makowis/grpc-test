## Go
ルートディレクトリ

```bat
cd golang
```

## Go Server Start

```bat
go run cmd/server/main.go
```

## Go Client Start

```bat
go run cmd/client/main.go
```

## Python

ルートディレクトリ

```bat
cd python
```

## Python Client

```bat
python3 client.py
```
